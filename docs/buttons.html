Buttons

Buttons are used to interact with the machine or system

Icon Buttons

Buttons that have an icon present with the text content to interact with the system

Link Buttons

Buttons that have an links present to redirect the browser to a new window , site or a section

Floating Action Buttons

These are =buttons to any special interaction like pick up a call , edit a note , add a new item or open messages to the user
