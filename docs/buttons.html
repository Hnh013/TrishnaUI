Buttons

A button is an element or a control that allows a user to directly communicate with the web application and send the necessary instructions to achieve a specific result.

Icon Buttons

These buttons have an icon present with the text content to better show the purpose or the entity the interact upon within the system, like lock screen options always have lock symbol, settings have a unique symbol to identify them as being settings of that particular section.

Link Buttons

These are the buttons that have links present inside them to redirect the browser to a new window , site or a section
