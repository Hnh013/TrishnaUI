Grids

A grid can be defined as an intersecting set of vertical and horizontal lines. CSS Grid layout separates a page into major sections. Grid property offers a grid-based layout system having rows and columns. Similar to the table, it enables a user to align the elements into rows and columns. We can define columns and rows on the grid by using grid-template-rows and grid-template-columns properties.
