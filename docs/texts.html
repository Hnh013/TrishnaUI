Typography

Typography includes everything the library offers or has set by default related to the fonts, i.e., it defines the way the text is formatted on the web page. It directly impacts the user as he/she writes or reads the text. It defines guidelines, rules and principles related to styles, sizes, etc. related to the text.

Font Family

Font family governs the style the text appears on the screen, how each and every character would appear.

Text Sizing

Text size is the size of character or font that appears on the screen. The default font size used is 1rem, that is equivalent to 16px. Besides the default there are many other font sizes available and can be used based on requirements of the user.
