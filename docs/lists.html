Lists

HTML lists are used to present list of information in well formed and semantic way. There are three different types of list in HTML and each one has a specific purpose and meaning.

Ordered Lists

Ordered lists are used when the order of the list's items is important. The list items in an ordered list are marked with numbers.

  1. Item One
  2. Item One
  1. Item Three
  2. Item Three
  1. Item Four
  2. Item Four
